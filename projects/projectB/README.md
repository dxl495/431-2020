## Video to help you deal with some common NHANES issues

- On 2020-11-17, Dr. Love built a video called [video_nhanes_example.mp4](https://drive.google.com/file/d/1ZCmMco_U4MXk2BDF3fQalzu1gM5revls/view?usp=sharing), available on our shared Google Drive in the Project B folder.
    - In the video, I download data using `nhanesA` from three data sets in NHANES 2017-18, select a few variables, merge the files, and then clean up one of the variables to turn "Don't Know" and "Refused" responses to missing values.
    - I made a mistake at the very end of the video when I suggested you ask questions via Canvas. Of course, I meant via [Piazza](https://piazza.com/case/fall2020/pqhs431).
- The [R Markdown file](https://github.com/THOMASELOVE/431-2020/blob/master/projects/projectB/video_nhanes.Rmd) I displayed and ran in the video is also available.
